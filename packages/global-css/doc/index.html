GLOBAL CSS

什么是 GLOBAL CSS？

Global CSS是micro-react的一部分，用来共享全局的CSS，作用于主应用(app-main)、子应用(app-*)及业务共享的组件库(ui-component)。 目的是在开发组件库、子应用亦或主应用时拥有统一的全局样式环境，避免由于开发环境样式的不一致导致的调整。 目前集成了 tailwindcss，可基于tailwindcss做快速的功能开发。
