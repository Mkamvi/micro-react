## Button 按钮

只需轻点按钮，用户就可以触发动作或做出选择。

按钮 传达了一系列用户可以执行的操作命令。 他们通常直接放置在您的用户界面中，例如：

- Dialogs 对话框
- Modal windows 模态窗口
- Forms 表单
- Cards 卡片
- Toolbars 工具栏
